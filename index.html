rintaun/homelab-charts

A Helm chart repository for self-hosted applications.

Usage

helm repo add rintaun https://rintaun.github.io/homelab-charts
helm repo update
helm search repo rintaun

Charts

Chart Version App version Description
actual-budget 0.1.1 26.4.0 Helm chart for Actual Budget — a local-first personal finance tool with optional OpenID Connect authentication.
beets 0.2.6 2.10.0 Helm chart for beets — the music geek's media organizer. Runs as a Kubernetes CronJob to periodically import and tag new music files.
navidrome 0.1.3 0.61.2 Helm chart for Navidrome — a self-hosted music server and streamer compatible with the Subsonic/Airsonic API.
papra 0.2.1 26.4.0 Helm chart for Papra — a document management system with ingestion folder watching and optional S3 document storage.
